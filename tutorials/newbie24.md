# Mutability

**Mutable** objects can be edited after being created.

**Immutable** objects cannot be edited after being created.


# Examples

| Immutable | Mutable      |
|-----------|--------------|
| strings   | lists        |
| integers  | dictionaries |
| floats    | sets         |
| tuples    |              |

# Why

**Immutable** objects are faster to access

**Mutable** objects easy to change
