# If / else

# Syntax

```python
if <conditional>:
    # do something if conditional is True
    <statement>
else:
    # do something if the conditional is False
    <statement>
```

# elif

```python
if <conditional>:
    # do something if conditional is True
    <statement>
elif <another-conditional>:
    # do something if another conditional is True
    <statement>
else:
    # do something if all conditionals are False
    <statement>
```
