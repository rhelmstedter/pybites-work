# Data Types

Python categorizes objects into different types.

# Data Types

| Data Type | Python Representation        |
|-----------|------------------------------|
| Text      | string                       |
| Numeric   | integer, float, complex      |
| Sequence  | list, tuple, range           |
| Mapping   | dictionary                   |
| Set       | set, frozenset               |
| Boolean   | boolean                      |
| Binary    | bytes, bytearray, memoryview |
